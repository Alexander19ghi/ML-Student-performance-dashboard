# Student Performance ML Dashboard

A beginner-friendly Streamlit dashboard that explains ML pipeline steps one by one:

1. Input Data
2. EDA
3. Data Cleaning and Engineering
4. Feature Selection
5. Data Split
6. Model Selection
7. Model Training
8. K-Fold Validation
9. Final Performance

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Community Cloud

1. Push this project to a GitHub repository.
2. Open https://share.streamlit.io
3. New app -> select repo -> branch `main` -> file `app.py`
4. Deploy.
